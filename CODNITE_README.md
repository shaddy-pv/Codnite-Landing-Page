# Codnite - Premium Cinematic Landing Page

A production-ready, high-end landing page for Codnite developer platform featuring smooth scroll animations, particle effects, and premium design.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion (animations)
- GSAP + ScrollTrigger (scroll animations)
- Lenis (smooth scrolling)

## Features

✨ Cinematic hero section with animated ember particles
🎬 Scroll-triggered animations throughout
🎨 Premium dark design with #FF6A00 accent color
📱 Fully responsive
⚡ Optimized performance (60fps)
🎯 Clean, scalable component structure

## Project Structure

```
src/
├── components/
│   ├── Hero.tsx           # Fullscreen hero with particle animation
│   ├── Problem.tsx        # Scroll-triggered problem statements
│   ├── ProductReveal.tsx  # 3D-style product showcase
│   ├── Features.tsx       # Interactive feature cards
│   ├── Experience.tsx     # Animated gradient section
│   ├── CTA.tsx           # Final call-to-action
│   └── Footer.tsx        # Clean footer
├── lib/
│   └── animations.ts     # Reusable animation configs
└── pages/
    └── Index.tsx         # Main landing page with Lenis smooth scroll
```

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
